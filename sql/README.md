# SQL Layer

This directory contains the SQL assets for the NBA Data Platform Reconstruction project.

## Planned Structure

| Folder | Purpose |
|---|---|
| schema | Database schema definitions |
| transformations | Data transformation logic |
| analytics | Analytical queries |
| views | Reusable analytical views |

## Goals

- Improve maintainability
- Reduce repetitive SQL
- Support analytical workloads
- Enable scalable querying
- Prepare for PostgreSQL migration
