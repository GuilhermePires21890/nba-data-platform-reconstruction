# Data Layer

This directory stores datasets and data lifecycle stages for the NBA Data Platform Reconstruction project.

## Planned Structure

| Folder | Purpose |
|---|---|
| raw | Original extracted datasets |
| processed | Cleaned and transformed data |
| curated | Analytics-ready datasets |
| legacy | Original academic project assets |

## Data Engineering Goals

- Preserve raw historical data
- Enable reproducible transformations
- Separate processing stages
- Improve analytical consistency
- Support scalable pipelines
