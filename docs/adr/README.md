# Architecture Decision Records (ADR)

This directory contains Architecture Decision Records for the NBA Data Platform Reconstruction project.

## Purpose

ADRs document important technical and architectural decisions made during the reconstruction and modernization process.

## Goals

- Preserve architectural reasoning
- Improve project maintainability
- Demonstrate engineering maturity
- Document trade-offs and rationale
- Improve portfolio quality

## Planned ADRs

|---|---|
| ADR-001 | Migration from legacy SQL Server structure to PostgreSQL |
| ADR-002 | Adoption of Python for ingestion and transformation |
| ADR-003 | Dataset consolidation strategy |
| ADR-004 | Dockerized local development environment |
| ADR-005 | Preservation of legacy academic assets |
